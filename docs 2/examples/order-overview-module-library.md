# What makes up this page

Think of these as building blocks.

You can mix, match, and reuse them across experiences.

---

## The core blocks

- Notifications  
- Account value  
- Goals  
- Activity  
- Help  
- Preferences  

---

## Why this matters

Instead of reinventing the page every time:

You reuse what already works.

That means:
- faster decisions  
- more consistency  
- less rework  

---

## The mindset shift

Stop thinking:

"What should this page look like?"

Start thinking:

"What does the user need, and what blocks support that?"

---

## How to use this

Pick the blocks you need.  
Put them in the right order.  

That's your page.
